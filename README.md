# Text Editor

* ## Project Summary

_This is simple text editor, base on NotePad. The purpose of this project is to edditiong text, using different fonds, color and e.t._
* ## Requirements

This program is written on C# 4.2.0 
Visual Studio 2022 
Windows Forms


* ## Usage

  ```
  File      Edit
  * New     * Undo          
  * Open    * Redo       
  * Save    * Cut
  * Exit    * Copy
            * Paste
            * Select All
            * DateTime
  ```
